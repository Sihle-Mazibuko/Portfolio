Interactive Design Process

The main goal when desinging this website was to creative a simple and fully comprehendable website, that proves efficent information without clouding the viewing with numerous amounts of content that they may not even know how to use and understand

The website has a single page, making it easy to flow through its numerous sections by scrolling down or using the functional buttons allowed at the top of the screen. This enforces two basic and evidently the same laws of navigating the website ensuring a user will not get lost when interacting with the website.

The website contains the following interface elements.

Buttons: There are a few buttons found on the website those that are included are found in the header and profile section and each serves to take the user to a specific be it an area in the website or external links.

Dropdowns: When scrolling down the website the header turns into a drop down that is always present at the top of the site displaying the navigation buttons of the website. On smaller screens navigation buttons are hidden and only the toggle dropdown is visible.

Icons: The only icons present in the website are the social media links and the drop down bar icon

The use of minimalistic interface elements is to give firstly a simple and minimalistic aesthetic to the website and secondly, because those are the only button necessary for the website.

Style guide

The aim is to create a minimalistic website

The use of a small range of colours (black,white,wheat, whitesmoke and sandy brown) help with this as the two none shades go well with one another being of similar shades and smokewhite adds a different feel from the ever-present white

Implementation notes

The arrangement of the content of the website follws the logic of seperating concerns, through putting things that are related to one another together

Semantics used are relevant to the content their display, as seen through the effecient use of "p tags" in this design section

There is no CSS or JavaScript in-line scripting as there is relevant linking to each necessary document
