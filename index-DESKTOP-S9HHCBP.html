Interactive Design Process

The main goal when designing this website was to create a simple and fully comprehendible website, that proves efficient information without clouding the viewing with numerous amounts of content that they may not even know how to use and understand.

Most of the websiteâ€™s content can be found on a single page, making it easy to flow through its numerous sections by scrolling down or using the functional buttons allowed at the top of the screen. This enforces two laws of navigating the website that ensure a user will not get lost when interacting with the elements. There are efficiently linked pages to longer pieces of text to ensure there is not too much infinite scrolling.

The website contains the following interface elements.

Buttons: There are a few buttons found on the website those that are included are found in the header and profile section and each serves to take the user to a specific be it an area in the website or external links.

Dropdowns: When scrolling down the website the header turns into a drop-down that is always present at the top of the site displaying the navigation buttons of the website. On smaller screens, navigation buttons are hidden and only the toggle dropdown is visible.

Icons: The only icons present in the website are the social media links and the drop-down bar icon.

The use of minimalistic interface elements is to give firstly a simple and minimalistic aesthetic to the website and secondly, because those are the only button necessary for the website.

Style guide

The aim is to create a minimalistic website

The use of a small range of colours (black #000000, white #FFFFFF, wheat #F5DEB3, white smoke #F5F5F5, grey #808080 and sandy brown #F4A460) help with this as the two none shades go well with one another being of similar shades and white smoke adds a different feel from the ever-present white. The grey acts as sort of a glue on the essay page as it can give a break from viewing either the black text or white smoke background for too long. The sans-serif font, namely Helvetica, is used throughout all text elements found on the website as sans-serifs are considered to be one most the least eye-straining and most legible fonts.

Implementation notes

The arrangement of the content of the website follows the logic of separating concerns, by putting things that are related to one another together.

Semantics used are relevant to the content their display, as seen through the efficient use of this design section and also by reducing the number of divs used and sticking to HTML semantics.

There is no CSS or JS in-line scripting as there is relevant linking to each necessary document.
